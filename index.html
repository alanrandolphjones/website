About me

Hello! My name is Alan.

I’m a web developer in Toronto, Canada with a love for functional programming and building web applications from the ground up. I currently work at Rich Media, where I help create digital tools and interactive projects for leading brands like Samsung, Scotiabank and Sun Life Financial.

I learned how to code as a student of the Front End Development Bootcamp at HackerYou. That’s where I learned the fundamentals of JavaScript and put together my first web application with React. I am currently enrolled in HackerYou’s Full-Stack Masterclass, learning how to work with Express and MongoDB. I also love film and pop culture, so in my off-hours you can probably find me watching Hackers or Mr. Robot.
